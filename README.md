# 图书馆系统 Web 前端

创作于大二上学期，这是我人生中第一次创作的成型前端界面系统。

## 我的收获

- 阿里巴巴 Ant Design 组件库：设计体验、规范、模块化代码开发的快感
- 对传统 Class 开发不满意，自学了 React Hooks
- 学会了处理 React 复杂状态管理的方法，理解 React 设计思想
- 了解 Creat React App 单页应用的开发模式、Webpack 的作用

## 存在的问题

- 接口设计不符合 RESTful 范式，缺乏管理
- fetch 调用
  - 重复 render 导致重复请求发起
  - 无法取消连接
